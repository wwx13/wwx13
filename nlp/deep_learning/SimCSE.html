# SimCSE

目标：无监督训练出更好的句向量

1.Dropout构建正样本

正样本在计算attention score时，会使用dropout，由于每次结果不同可以当做增广正样本。

这样的样本模型应该认为是相似的。

实际训练时样本：

一个正样本和一个正样本加若干负样本组成。目标为计算目标正样本和（一个正样本加若干负样本）的相似度，并计算交叉熵，正样本和正样本的相似度尽量高， 负样本和正样本原理， 负样本均匀。

2.bert等多个模型做特征提取

3.损失就是交叉熵损失

# 效果

SimCse效果在_ATEC/BQ/LCQMC__上提升个20点，以bert为基线。

# 为什么好

对于句向量来说， 好的句向量第一具有好的alignment。就是正负样本区分开好。

第二具有好的Uniformity, 样本竟可能均匀在区域中。SimCse就对这个有优势，bert的句向量偏向有好的alignment，但是没有好的Uniformity。
