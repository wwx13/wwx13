# word2vec

定义一个词有两个向量表示，一个是中心词向量，一个是背景词向量。

# skip gram

在一个固定长度窗口内，用一个中心词预测所有背景词的条件概率极大似然估计。 背景词之间条件独立。

最后使用中心词向量代表词。

# cbow

在一个固定长度窗口内，用若干背景词预测中心词的条件概率极大似然估计。 其中条件概率里用这 些背景词向量的平均代表综合的背景词信息。

最后使用背景词向量代表词。

skip gram 和cbow都是谁是条件谁最后用于代表词。

# 近似训练

由于两种概率模型求解中，关于变量的梯度涉及词汇表大小的条件概率，因此需要考虑高效的训练策略。 下面两种都是修改了原来的概率模型， 而不是遵从上面的概率模型。

# hierarchy softmax

把给定中心词分类背景词考虑为 给定中心词逐步划分到背景词的过程。给定中心词起始背景词的解集为全集，接着二分选择合适的解集 一直到背景词。这里解空间的划分是提前确定好的，所以是要求训练的模型满足解空间的划分。划分解空间就是构建一个二叉树，叶节点为一个词， 非叶节点为背景词向量，
。由于我们的条件概率是已知背景词是哪个叶节点，因此我们提前确定了一个路径，我们需要反映在模型定义是，模型按照划分左右进行调整，并且左右应该概率之和为1。 因此sigmoid(-1或+1 * (当前非叶节点(背景词向量) * 中心词向量))代表一个给定中心词向量在当前子空间正确选择下一个空间的概率。

设每步划分都是独立的因此连乘就是给定中心词预测背景词的概率表达。 复杂度为log(v). v为词表大小。

# negtive sampling

把给定中心词分类背景词概率修改为 中心词和背景词出现（1个正例）并且k个负例出现(中心词和某个随机的背景词)。

一个完整的训练样本为 一个中心词+(window-1)个背景词 + (window-1)个背景词 * K个噪声词
损失为 binary cross entrophy loss: (k+1)个二分类问题

我们将与中心词距离不超过背景窗口大小的词作为它的背景词。下面定义函数提取出所有中心词和它们的背景词。它每次在整数1和max_window_size（最大背景窗口）之间随机均匀采样一个整数作为背景窗口大小。

def get_centers_and_contexts(dataset, max_window_size):
    centers, contexts = [], []
    for st in dataset:
        if len(st) < 2:  # 每个句子至少要有2个词才可能组成一对“中心词-背景词”
            continue
        centers += st
        for center_i in range(len(st)):
            window_size = random.randint(1, max_window_size)
            indices = list(range(max(0, center_i - window_size),
                                 min(len(st), center_i + 1 + window_size)))
            indices.remove(center_i)  # 将中心词排除在背景词之外
            contexts.append([st[idx] for idx in indices])
    return centers, contexts


我们使用负采样来进行近似训练。对于一对中心词和背景词，我们随机采样K个噪声词（实验中设K=5）。根据word2vec论文的建议，噪声词采样概率P(w)设为w词频与总词频之比的0.75次方 [2]。

def get_negatives(all_contexts, sampling_weights, K):
    all_negatives, neg_candidates, i = [], [], 0
    population = list(range(len(sampling_weights)))
    for contexts in all_contexts:
        negatives = []
        while len(negatives) < len(contexts) * K:
            if i == len(neg_candidates):
                # 根据每个词的权重（sampling_weights）随机生成k个词的索引作为噪声词。
                # 为了高效计算，可以将k设得稍大一点
                i, neg_candidates = 0, random.choices(
                    population, sampling_weights, k=int(1e5))
            neg, i = neg_candidates[i], i + 1
            # 噪声词不能是背景词
            if neg not in set(contexts):
                negatives.append(neg)
        all_negatives.append(negatives)
    return all_negatives

def batchify(data):
    max_len = max(len(c) + len(n) for _, c, n in data)
    centers, contexts_negatives, masks, labels = [], [], [], []
    for center, context, negative in data:
        center=center.numpy().tolist()
        context=context.numpy().tolist()
        negative=negative.numpy().tolist()
        cur_len = len(context) + len(negative)
        centers += [center]
        contexts_negatives += [context + negative + [0] * (max_len - cur_len)]
        masks += [[1] * cur_len + [0] * (max_len - cur_len)]
        labels += [[1] * len(context) + [0] * (max_len - len(context))]
    return tf.data.Dataset.from_tensor_slices((tf.reshape(tf.convert_to_tensor(centers),shape=(-1, 1)), tf.convert_to_tensor(contexts_negatives),
            tf.convert_to_tensor(masks), tf.convert_to_tensor(labels)))
