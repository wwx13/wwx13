# KD树

  1. kd树其实是一个二叉搜索树。但是为什么放在机器学习里呢？因为在某些机器学习方法里会 使用kd树来实现。例如KNN。

# KNN

# 思想

预测时，输入一个向量在已有的实例向量中找到最近距离的k个，利用某些决策规则如 多数表决策略， 这K个最多的类别 就赋予输入向量这个类别。训练时，把类心的类别作为这个cell的类别。当K=1，我们称为最近邻。K近邻实际上是对训练集的向量进行特征空间的划分打上类别，并作为分类的模型。

# 三个问题

  1. K值选择 交叉验证法
  2. 距离度量 knn
  3. 决策规则 KD树的建立类似于数据的训练，建立树就建立了模型， KD树的搜索就是预测。搜索的目的是搜索最近的K个节点。
  • 构建KD树

  1. 选取方差最大的特征作为分割特征；

  2. 选择该特征的中位数作为分割点；

  3. 将数据集中该特征小于中位数的传递给根节点的左儿子，大于中位数的传递给根节点的右儿子；

  4. 递归执行步骤2-4，直到所有数据都被建立到KD Tree的节点上为止。

  • 搜索KD树

KD Tree建好之后，接下来就要利用KD Tree对元素进行查找了。查找的方式在BST的基础上又增加了一些难度，如下：

下面示例最近邻的搜索k=1。

  1. 从根节点开始，根据目标在分割特征中是否小于或大于当前节点，向左或向右移动。

  2. 一旦算法到达叶节点，它就将节点点保存为“当前最佳”。

  3. 回溯，即从叶节点再返回到根节点

  4. 如果当前节点比当前最佳节点更接近，那么它就成为当前最好的。

  5. 如果目标点距离 当前节点的父节点的分割超平面 小于 目标点距离当前节点（即当前最近距离），说明当前节点的兄弟节点所在的子树有可能包含更近的点。因此需要对这个兄弟节点递归执行1-4步

# 说明步骤5
如果目标点距离 当前节点的父节点的分割超平面 小于 目标点距离当前节点，
等价于分割维度取值向超平面另一侧移动距离变小。
