# 总述

svm 用于分类问题，思想主要是希望找到一个支持超平面，使得分离所有的点，且分离的最开。 这个最开是指里超平面最近的点是尽可能的远。术语化：svm目标是可行集里的支持向量距离支撑超平面最远。 接下来需要研究，如何计算距离超平面，如何定义我们的目标和约束，并求解。

# 点到超平面距离

  1. 绝对距离与几何间隔 绝对距离：代表预测的置信度和点预测的正负信息。点到超平面的绝对远近是带方向的，方向为点落在半空间代表的正负。 几何间隔： 绝对距离基础上乘以点的真实标签，代表预测结果的正误和预测置信度。
